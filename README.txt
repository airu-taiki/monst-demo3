GitHub Pages 公開手順

1. zipを展開
2. GitHubで新規リポジトリを作成
3. 中身をそのままアップロード
4. Settings → Pages → Deploy from a branch
5. Branch: main / Folder: /(root)
6. 数十秒待ってURLを開く

使い方
- index.html がトップページです
- 最初のクリックで演出が始まります
- 録画して編集素材として使ってください

メモ
- alert.mp4 と bgm.mp4 は音あり
- glitch.mov は見た目用オーバーレイです
- GitHub Pagesは最初のユーザー操作がないと音付き自動再生が止まることがあります
